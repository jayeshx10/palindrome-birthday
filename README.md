# Palindrome birthday checker

### ABOUT THE PROJECT:

  A web app that tells you if your birthday is palindrome or not. If not it finds out the nearest one.

### BUILT WITH : 
  
  HTML, CSS and VanillaJS. 
  
### LIVE LINK:
